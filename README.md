# Task Manager App

A Task Management System built with Next.js, Tailwind CSS, and NextAuth authentication. This application allows users to perform CRUD (Create, Read, Update, Delete) operations on tasks, providing a streamlined way to manage their to-do lists. Users can toggle between incomplete and completed tasks, ensuring efficient organization and productivity.

## Description
The Task Manager App is a  solution for managing tasks efficiently. Built on the Next.js framework with Tailwind CSS for styling and NextAuth for authentication, this application provides a seamless experience for users to create, edit, and delete tasks. The ability to toggle between incomplete and completed tasks offers enhanced organization and clarity.

## Features
- **User Authentication:**
  Secure user authentication powered by NextAuth, ensuring a safe and personalized experience.
- **Task Operations:**
  Perform CRUD operations on tasks, including creating new tasks, updating existing ones, and marking tasks as completed.
- **Toggle Between Incomplete and Completed Tasks:**
  Easily switch between incomplete and completed tasks for better task organization.
- **Responsive Design:**
  A responsive design that ensures a consistent and user-friendly experience across different devices.

## Technologies Used
- Next.js 13
- Tailwind CSS
- NextAuth Authentication
- MongoDb (for data storage)
- Mongoose (database toolkit)

## Installation
To run the Task Manager App locally, follow these steps:
1. Clone the repository: `https://github.com/Emmyojile/Task-Manager.git`
2. Navigate to the project directory: `cd task-manager-app`
3. Install dependencies: `npm install`
4. Set up the database and configure environmental variables as needed.
5. Run the application: `npm run dev`

## Usage
1. Register or log in using NextAuth authentication.
2. Explore the intuitive task management interface.
3. Create, update, and delete tasks based on your needs.
4. Toggle between incomplete and completed tasks for better organization.
